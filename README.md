Интерактивный генератор спектров огибающей с дефектами подшипников.

## 📖 Что умеет

- 🎛️ Генерация спектров огибающей с дефектами BPFO, BPFI, BSF, FTF
- 📊 4 графика: временной сигнал, огибающая, спектры (линейный и логарифмический)
- 🔧 Выбор фильтра: 1/3 октавы или 1 октава
- 📈 Выбор центральной частоты fc из рекомендуемых значений
- 🎓 Игра "Угадай дефект" для обучения

## 📚 База подшипников

Встроены 10 популярных подшипников: 6205, 6208, 6305, 6308, 6310, 6312, 22210, 22220, 30210, 7014
